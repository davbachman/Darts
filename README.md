# Darts

[Open the app](https://davbachman.github.io/Darts/)

Created by David Bachman with GPT 5.5. To learn more about David see https://pzacad.pitzer.edu/~dbachman/, and subscribe to his AI blog *Entropy Bonus* at https://profbachman.substack.com/.

## Brief description
A browser-based darts throwing app set in a pub, with hand-tracked dart throws, a realistic dartboard, and chalkboard-style scoring for Practice, 301, Cricket, and Around the Clock.

## Instructions for use
Choose a game mode, allow camera access, pinch thumb and index finger to grab a dart, push toward the screen, and release to throw. Use the chalkboard sidebar to follow the score and reset or return to the menu.
